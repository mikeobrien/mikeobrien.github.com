--- layout: post title: Moving the ThoughtWorks Cruise Configuration File on Windows tags: [ThoughtWorks Cruise] ---

Relocating the ThoughtWorks Cruise configuration file is pretty simple. We have our build artifacts located on a separate partition and we wanted the Cruise config file to be located there also.

  1. Open the server service config file: [Program Files]\Cruise Server\config\wrapper-server.conf
  2. Change the location of the "cruise.config.file" parameter to the new location of the config file.



  3. Copy the cruise-config.xml file from the same directory as the server service config file above to the desired location.
  4. Restart the Cruise server service.
