--- layout: post title: Configuring WCF 3.0 to Run Under ASP.NET tags: [.NET 3.0, Configuration, ASP.NET, WCF] ---

In order to configure WCF 3.0 to run under ASP.NET you need to run the installer (ServiceModelReg.exe) in the WFC 3.0 Framework folder (\\Microsoft.NET\Framework\v3.0\Windows Communication Foundation) with the /i parameter.
