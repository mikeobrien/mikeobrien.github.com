Whenever I regen my machine I always run into the asinine default IE security settings as the "Internet" zone defaults to "High" security. The "High" setting makes IE pretty much useless, disabling JavaScript and a ton of other things. Not to mention all the stupid security warnings that pop up in both IE and Windows Explorer asking you to add a million domains to the trusted sites and warning you that files may be unsafe. Under the administrator account, the "Internet" zone is configurable (Although not with the slider, which is annoying), but for new users the controls are completely disabled for this zone so you cant even change them:

When I regen I create a new windows user to use on the box (I don't use the administrator account) so I run into this every time. And yes I am a Chrome user, but the download page for Chrome requires that JavaScript be enabled so its a bit of a catch 22. In any event, if you want to enable all the controls for the "Internet" zone, follow the instructions here (Thanks Mike for pointing this out). On thing that the reg below does though is set the lowest possible setting for the slider to Low whereas the method in the link only allows you to drop down only to Medium. Prolly not a big deal though. run the following reg file:

Windows Registry Editor Version 5.00

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings\Zones\3]
"Flags"=dword:00000003

[HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Internet Settings\Zones\3]
"MinLevel"=dword:00010000

[HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Internet Settings\ZoneMap]
"IEHarden"=dword:00000000

This will enable all the controls (Including the slider) so you can configure the security to your hearts content:
